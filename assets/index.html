CVE Vulnerability Scanner

Scan your Maven, Node.js, Go and Python projects for known security vulnerabilities (CVEs) in dependencies.

üîÑ Auto-detect: Automatically detects project type and uses the best scanner.
  • Maven projects (pom.xml) ‚Üí OWASP Dependency-Check
  • Node.js projects (package-lock.json) ‚Üí npm audit
  • Yarn projects (yarn.lock) ‚Üí yarn audit
  • pnpm projects (pnpm-lock.yaml) ‚Üí pnpm audit
  • Go projects (go.mod) ‚Üí govulncheck
  • Python projects (requirements.txt) ‚Üí pip-audit
  • PHP projects (composer.json) ‚Üí composer audit
‚òï OWASP Dependency-Check: Uses Maven plugin, no additional install required. Comprehensive CVE database coverage. First scan may take longer to download vulnerability database.
